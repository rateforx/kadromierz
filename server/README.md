# Kadromierz

Developed by Filip Janta

## Server

1. Go to the server directory: 

    `.../kadromierz-master/server/`

2. Install the dependencies, run:

    `yarn` or `npm i`

3. Start the server:

    `yarn start` or `npm run start`

## React App

To run the app you need to install the dependencies and run the development environment by repeating the same steps like you did for the server. 

1. Go to the project directory:

    `.../kadromierz-master/`
    
2. Run `yarn`, then `yarn start`.

3. Open up your browser and visit `http://localhost:3000` to see the page.
